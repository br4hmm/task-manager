## Routes

| About         | HTTP Methods               |
| ------------- | -------------------------- |
| Get all tasks | `GET /api/v1/tasks`        |
| Create task   | `POST /api/v1/tasks`       |
| Get task      | `GET /api/v1/tasks/:id`    |
| Update task   | `PATCH /api/v1/tasks/:id`  |
| Replace task  | `PUT /api/v1/tasks/:id`    |
| Delete task   | `DELETE /api/v1/tasks/:id` |
